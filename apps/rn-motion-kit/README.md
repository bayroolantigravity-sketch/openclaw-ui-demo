# RN Motion Kit (Expo SDK 52)

21st.dev tarzı his veren React Native motion başlangıç kiti.

## İçerik

- Reanimated ile **stagger card reveal**
- Press feedback (scale + haptic)
- Cinematic native-stack page transition
- Motion bottom sheet (backdrop fade + spring slide)

## Kurulum

```bash
cd apps/rn-motion-kit
npm install
npm start
```

## Not

Reanimated plugin zaten `babel.config.js` içinde aktif.
